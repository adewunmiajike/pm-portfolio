# Metrics Alerts & Thresholds

## Daily Monitoring

| Metric | Green | Yellow | Red | Action |
|--------|-------|--------|-----|--------|
| DAU | >3,000 | 2,000-3,000 | <2,000 | Check product issues |
| Output Accuracy | >80% | 75-80% | <75% | Review prompts, run evals |
| Churn Rate | <5% | 5-7% | >7% | Analyze cohort, email campaign |
| API Uptime | >99.5% | 99-99.5% | <99% | Page on-call engineer |

## Weekly Review
- Compare vs 4-week average
- Check leading indicators
- Plan fixes for any red zones
- Update stakeholders

## Monthly Planning
- OKR status review
- Competitive benchmarking
- Strategic adjustments
- Board reporting
