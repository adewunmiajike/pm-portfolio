# Churn Analysis

## Problem Statement
Analyzing why customers churn after their first week. Finding root causes and recommending fixes.

## Methodology
- Cohort analysis: Compare retention by signup date
- Behavioral segmentation: Analyze purchase patterns
- Time-to-value: Track when users take key actions

## Key Findings
- Same-day purchasers: 6.7% churn
- 8+ day purchasers: 80% churn
- Habit formation critical: 8+ orders = 0% churn

## Recommendations
1. Improve first-day experience (15% impact)
2. Re-engagement emails (10% impact)
3. Build habit loop (20% impact)
